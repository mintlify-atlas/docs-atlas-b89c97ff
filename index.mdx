## What you can build

Go Payment Dashboard is a production-quality reference implementation showing how to structure a full-stack Go application with clean architecture, type-safe SQL, and a modern React frontend.

<CardGroup cols={3}>
  <Card title="Go REST API" icon="server">
    chi router, CORS middleware, structured JSON responses, and request validation.
  </Card>
  <Card title="React frontend" icon="browser">
    Vite + TypeScript, TanStack Query, Tailwind CSS, and shadcn/ui components.
  </Card>
  <Card title="PostgreSQL" icon="database">
    SQL migrations, sqlc-generated type-safe queries, and pgx connection pooling.
  </Card>
  <Card title="Docker Compose" icon="docker">
    Single-command local development with Air live-reload for the Go server.
  </Card>
  <Card title="GitHub Actions" icon="github">
    Automated Go tests on every PR and Fly.io deployment on merge to main.
  </Card>
  <Card title="Testcontainers" icon="flask">
    Integration tests that spin up a real PostgreSQL container automatically.
  </Card>
</CardGroup>
